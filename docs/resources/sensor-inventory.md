## Port Assignment Guide

### Digital Ports (D2-D13)

- **D2-D8:** General digital I/O, some support PWM
- **D3, D5, D6, D9, D10, D11:** PWM-capable (LED brightness, servo)
- **D13:** Built-in LED (can also be used for external devices)

### Analog Ports (A0-A5)

- **A0-A3:** Analog input (0-5V ‚Üí 0-1023 or 0-16383 on Uno R4)
- **A4 (SDA), A5 (SCL):** I2C communication (also available as digital)

### I2C Ports

- **Dedicated I2C grove connector** on Base Shield
- Multiple I2C devices share same bus (different addresses)
- Use I2C Hub for more connections

### Special Considerations

- **PWM:** Required for LED dimming, servo control
- **Digital Pulse:** Used for time-sensitive devices (ultrasonic, buzzer, displays)
- **I2C:** All I2C devices connect to same bus via hub if needed
